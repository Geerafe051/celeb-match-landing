# Celeb Match landing

Статичный лендинг для сервиса гостевого продюсирования.

## Локальный тестовый контур

Запуск:

```bash
python3 -m http.server 8080
```

Открыть:

```text
http://localhost:8080/
```

На `localhost` форма работает в тестовом режиме: заявка не отправляется в Google Sheets,
а сохраняется в `localStorage` браузера под ключом `celebMatchTestLeads`. Это позволяет
проверять верстку, валидацию и сообщение об успешной отправке без мусора в боевой таблице.

Чтобы из локального контура отправить заявку в Google Sheets, откройте:

```text
http://localhost:8080/?send=1
```

## Продакшен

Продакшен публикуется через GitHub Pages из ветки `main`.

Перед публикацией проверить:

- ссылки в футере ведут на `privacy.html` и `personal-data.html`;
- `GOOGLE_SCRIPT_URL` в `script.js` указывает на актуальный Google Apps Script Web App;
- форма отправляет заявку и показывает сообщение об успешной отправке;
- документы по персональным данным заполнены реальными реквизитами оператора.
